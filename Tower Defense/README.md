# Hex Tower Defense

육각형 격자 위에서 적의 러시를 막아내는 완성된 타워 디펜스 게임이다. 단 한 번 뚫리면 기지 체력이 줄어들며, 모든 웨이브를 막아내면 승리한다. 직관적인 조작과 빠른 전투 루프가 핵심이다.

## 게임 구성

- **맵과 경로**: 7타일 반경의 육각형 맵. 적은 무작위로 선택된 외곽에서 중앙 기지까지 직선 경로로 진입한다.
- **웨이브 진행**: 총 600웨이브. 웨이브가 끝나면 짧은 휴식 시간이 주어지며 `N` 키로 다음 웨이브를 수동 시작한다.
- **기지**: 중앙 타일에 위치하며 기본 체력은 20. 적이 도달할 때마다 1씩 감소한다.
- **경제**: 적 처치 시 코인을 획득하며 기본 자본은 200. 건설·업그레이드·치유에 사용한다.

## 타워

| 이름 | 특징 |
| --- | --- |
| Basic | 저렴한 범용 포탑. 사거리 170, 단일 투사체 공격.
| Sniper | 긴 사거리와 높은 피해를 지닌 저속 포탑.
| Rapid | 짧은 사거리 대신 빠른 연사로 묶여 오는 적을 녹인다.
| Wall | 공격하지 않지만 느리게 만드는 오라를 제공하고 내구도가 높다.

- 타워는 클릭하거나 `Space`/`Enter`로 건설하며, 이미 지어진 타워를 다시 선택하면 업그레이드된다.
- 업그레이드 시 비용이 소폭 상승하며 사거리·피해량·연사력이 개선된다. 최대 5레벨.
- 휴식 시간 동안 `RMB` 키로 선택한 타워를 코스트를 지불해 회복할 수 있다.

## 적

- **Grunt**: 기본 적. 느린 속도와 보통 체력, 근·원거리 공격을 모두 보유.
- **Swift**: 이동 속도가 빠른 돌격형. 체력은 낮지만 초반 방어선을 뚫기 쉽다.
- **Tank**: 느리지만 높은 체력과 강한 근접 피해를 지닌 적.

적은 웨이브 번호에 비례해 체력·보상·속도가 점진적으로 상승한다. 벽(Wall) 타워의 감속 효과는 적 이동 속도를 최소 10%까지 낮춘다.

## 조작법

- **건설 선택**: 숫자 `1-9`로 선택, 마우스 클릭 혹은 `Space/Enter`로 배치.
- **타일 탐색**: 마우스 이동으로 하이라이트, 혹은 방향키/`WASD`/`Q`/`E`로 선택 이동.
- **웨이브 제어**: 준비 시간 이후 `N` 키로 수동 시작.
- **치유**: 휴식 시간 중 `H`로 선택한 타워 회복.
- **메뉴 이동**: 시작 화면에서 `Start` 버튼, 규칙은 `Rules`, 게임 종료 후 `Main Menu`로 복귀.

## 오디오

- **인트로/아웃트로**: `start-272637` 트랙을 메인 메뉴와 결과 화면에 사용한다.
- **게임플레이**: `warrior-defense-fighting-music-335681` 트랙을 웨이브 중 루프 재생한다.

두 오디오 파일을 `Tower Defense` 디렉터리(또는 실행 디렉터리) 아래에 같은 이름의 MP3 파일로 배치하면 자동으로 불러온다. 파일이 없거나 로드에 실패하면 게임은 조용히 진행된다.

## 실행

```bash
pip install -r ../requirements.txt
python main.py
```

메뉴 배경음이 재생되는지 확인하려면 오디오 장치와 해당 트랙 파일이 필요하다.
